# Atelier E-Commerce

## Overview
Atelier is a premium, fully responsive e-commerce experience built with plain HTML, CSS, and vanilla JavaScript. It uses the DummyJSON products API and includes modern shopping flows for browsing, searching, saving favorites, cart management, and checkout.

## Folder Structure
- index.html — Home page
- product.html — Product listing and filtering experience
- cart.html — Cart and order summary
- wishlist.html — Saved products
- checkout.html — Shipping and payment form
- css/ — Global and page-specific styles
- js/ — Modular JavaScript modules for app logic
- assets/ — Icons and images

## Features
- Responsive premium UI
- Live product search and filtering
- Product cards with wishlist and cart actions
- Local storage persistence for cart, wishlist, theme, and recent views
- Modern loaders, toasts, and smooth interactions
- Checkout flow with order confirmation

## How to Run
1. Open the project folder in a browser, or run a simple local server such as:
   - `python -m http.server 8000`
2. Visit `http://localhost:8000/`

## API Used
- DummyJSON Products API: https://dummyjson.com/products
